Features

Blistering Fast

Built in Rust, Vector is fast, lean, and efficient. It's designed to handle the most demanding environments.

Vendor Agnostic

Vector is vendor agnostic, unbiased, and obosessed with helping you get the most of your data. Regardless of the tool.

Agent or Service

Vector aims to be the single, and only, tool needed to get data from A to B, deploying as an agent or service.

Logs or Metrics

Vector unifies logs and metrics, seamlessly converting logs to metrics. No need for 2 separate tools.

Programmable Transforms

An embedded LUA engine makes it easy to program powerful transforms, no limitations.

At-Least Once Delivery

Vector's core competency is delivery, it's not just a means to an end for us, which is why we offer an at least once guarantee.
